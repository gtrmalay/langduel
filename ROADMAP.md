## Этап 1: ELO + Лидерборд

### БД
```sql
CREATE TABLE user_ratings (
  user_id UUID PRIMARY KEY REFERENCES users(user_id),
  elo INT DEFAULT 1000,
  rank VARCHAR(20) DEFAULT 'newbie',
  games_played INT DEFAULT 0,
  wins INT DEFAULT 0,
  losses INT DEFAULT 0,
  current_streak INT DEFAULT 0,
  best_streak INT DEFAULT 0,
  total_losses_at_zero INT DEFAULT 0,
  updated_at TIMESTAMPTZ DEFAULT NOW()
);
```

### Ранги
| Звание | ELO | Условие |
|--------|-----|---------|
| 🥉 Newbie | 0-999 | Все новые |
| 🥈 Apprentice | 1000-1999 | - |
| 🥇 Expert | 2000-2999 | - |
| 💎 Master | 3000+ | - |
| 😔 Struggler | 0 | >10 поражений при 0 ELO |

### Формула
- Победа: +25 ELO
- Поражение: -15 ELO
- Минимум: 0

### API
- `GET /leaderboard` - топ-100 (без гостей)
- `GET /me/rating` - рейтинг текущего юзера

### UI
- `/leaderboard` - страница лидерборда
- ELO в профиле + полоса прогресса
- ELO в HUD боя

## Этап 2: AI Генерация

### Направления перевода
1. EN → RU
2. RU → EN

### При создании комнаты
```
Тема: [Фильмы ▼]        (movies, travel, food, sports, slang)
Сложность: [Средняя ▼]   (beginner, intermediate, advanced)
Язык: [EN → RU ▼]

[⚡ СГЕНЕРИРОВАТЬ]  ← AI генерирует 20-30 фраз
```

### AI Провайдер
- Grok (xAI) - рекомендуется ($5/месяц)
- OpenAI GPT-3.5 - альтернатива

### Формат хранения в БД
```json
{
  "prompt": "How's it going?",
  "answers": ["как дела", "как идёт", "как сам", "как поживаешь"],
  "topic": "movies",
  "difficulty": 2
}
```

### Промпт
```
Сгенерируй 25 фраз для языковой дуэли.
Тема: {topic}
Сложность: {difficulty}
Язык: EN → RU

Формат JSON:
[{"prompt": "...", "answers": ["...", "...", "..."]}]

- Фразы реальные, частые
- Для advanced включай сленг
- 3-5 вариаций ответа на фразу
```
